# Site AdaptSW

Site statique présentant le TIPE AdaptSW de Youssef Litayem.


L’idée est de comprendre comment un message, comme un SOS, peut avancer de téléphone en téléphone grâce aux déplacements des personnes, pratique en situation d'urgence, catastrophes naturelles et destruction des infrastructures.

Le projet aboutit à AdaptSW, un protocole que j’ai construit à partir de Spray-and-Wait. Son but est de choisir automatiquement combien de copies d’un message il faut créer cad assez pour augmenter les chances de livraison, mais pas trop pour éviter les transmissions inutiles.
Le site reprend le contenu du TIPE sous une forme plus claire qu’un PDF classique, avec les explications, les formules, le modèle mathématique, le fonctionnement du protocole et l’implémentation Android. 


## Remarque sur la simulation

La visualisation animée sert à expliquer le fonctionnement de Spray-and-Wait et d'AdaptSW. Elle n'est pas présentée comme un résultat expérimental et ne remplace pas les expériences menées avec les téléphones Android.
